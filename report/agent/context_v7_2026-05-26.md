## Key Mental Model Mappings (React Native → SwiftUI) [[CHANGED]]

| React Native / TypeScript | Swift / SwiftUI |
|---|---|
| `const` | `let` |
| `let` | `var` |
| `interface` / object type | `struct` |
| Service / store class | `final class` |
| `Intl.NumberFormat` / `toLocaleString` | `NumberFormatter` / `Double.asVND()` |
| `string \| undefined` | `String?` |
| `value ?? fallback` | `value ?? fallback` |
| `if (!x) return` | `guard let x else { return }` |
| `user?.profile?.avatar` | `user?.profile?.avatar` |
| `T[]` / `Array<T>` | `[T]` |
| `arr.map(x => x.amount)` | `arr.map { $0.amount }` |
| `Record<K, V>` | `[K: V]` |
| `obj[k] ?? 0` (accumulate) | `dict[k, default: 0] += v` |
| String union / const enum | `enum` + `Equatable` |
| Redux selector | Pure fn `filterTransactions(_:by:)` |
| `onPress` / `onSuccess` / `onComplete` | Closure / trailing closure |
| `Button` children + `onPress` | `Button { action } label: { View }` |
| TypeScript `interface` | Swift `protocol` |
| `implements Interface` | `: Protocol` conformance |
| Generic `<T>` | `struct ApiResponse<T>` |
| Mock repository / service | Class conforming to protocol |
| Inject dependency | `func foo(repository: TransactionRepository)` |
| `async function` | `func ... async` |
| `await` | `await` |
| Promise rejection | `throws` / `TransactionError` |
| `try/catch` | `do/try/catch` |
| Fire-and-forget async | `Task { }` |
| `error.message` (UI) | `LocalizedError.errorDescription` |
| Sync cache read | `TransactionRepository.getTransactions()` |
| Async API call | `TransactionService.fetchTransaction` `async throws` |
| Function component | `struct SomeView: View` |
| Props | Stored properties (`let`) |
| `useState` | `@State` |
| Controlled prop | `@Binding` |
| Context | `@Environment` / `@EnvironmentObject` |
| Redux/Zustand store | `@Observable` class |
| `useEffect(() => {}, [])` | `.task` / `.onAppear` |
| `useEffect(..., [dep])` | `.onChange(of:)` |
| React Navigation | `NavigationStack` |
| Modal | `sheet` / `fullScreenCover` |
| FlatList | `List` / `ScrollView + LazyVStack` |

## Common Mistakes to Watch For [[CHANGED]]

1. Đặt side effects trong `body` thay vì dùng `.task`
2. Cố dùng Flexbox thay vì `VStack/HStack/ZStack`
3. Dùng global store quá nhiều thay vì `@State` / `@Binding` local
4. Map hooks 1-1 sang SwiftUI (không có mapping hoàn hảo)
5. Customize `List` quá nhiều — dùng `ScrollView + LazyVStack` thay thế
6. Dùng `.formatted()` khi cần format cố định — nhớ `NumberFormatter` hoặc `Locale` explicit
7. Force unwrap `!` trong production code — dùng `if let` / `guard let` / `??`
8. Nhầm `dict[key, default: 0]` với optional `note ??` — khác ngữ cảnh
9. Truyền `note ?? ""` vào hàm cần `String?` — làm `displayNote` không fallback đúng
10. Đặt tên protocol method số ít — dùng plural `getTransactions` / `fetchTransactions`
11. Gộp nhiều test `throws` trong một `do-catch` — lỗi đầu làm code phía sau không chạy [[CHANGED]]
12. Playground: `let service` top-level + `Task` nonisolated — warning Main Actor; tạo service trong `Task` hoặc `Task { @MainActor in }` [[CHANGED]]
