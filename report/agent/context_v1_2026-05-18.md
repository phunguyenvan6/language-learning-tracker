## Key Mental Model Mappings (React Native → SwiftUI)

| React Native / TypeScript | Swift / SwiftUI |
|---|---|
| Function component | `struct SomeView: View` |
| Props | Stored properties (`let`) |
| `useState` | `@State` |
| Controlled prop | `@Binding` |
| Context | `@Environment` / `@EnvironmentObject` |
| Redux/Zustand store | `@Observable` class |
| `useEffect(() => {}, [])` | `.task` / `.onAppear` |
| `useEffect(..., [dep])` | `.onChange(of:)` |
| React Navigation | `NavigationStack` |
| Modal | `sheet` / `fullScreenCover` |
| FlatList | `List` / `ScrollView + LazyVStack` |
| TypeScript interface | Swift `protocol` |
| Async/await | Swift `async/await` |
| `try/catch` | `do/try/catch` |

---

## Common Mistakes to Watch For

1. Đặt side effects trong `body` thay vì dùng `.task`
2. Cố dùng Flexbox thay vì `VStack/HStack/ZStack`
3. Dùng global store quá nhiều thay vì `@State` / `@Binding` local
4. Map hooks 1-1 sang SwiftUI (không có mapping hoàn hảo)
5. Customize `List` quá nhiều — dùng `ScrollView + LazyVStack` thay thế
