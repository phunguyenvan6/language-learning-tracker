## Key Mental Model Mappings (React Native -> SwiftUI) [[CHANGED]]

| React Native / TypeScript | Swift / SwiftUI |
|---|---|
| `const` | `let` |
| `let` | `var` |
| `interface` / object type | `struct` |
| Redux/Zustand store | `@Observable` class (Observation) |
| `useState(store)` + mutate | `@State` owner + `@Observable` property tracking |
| Controlled component | `@Binding` — parent `$field`, child `@Binding var field` |
| `React.createContext` + `Context.Provider` | `@State` at App + `.environment(store)` |
| `useContext(MyContext)` | `@Environment(MyType.self)` |
| `useEffect(() => {}, [])` async | `.task { }` |
| `useEffect(..., [value])` | `.onChange(of:)` |
| Callback prop | Closure `(String) -> Void` |
| Stack Navigator | `NavigationStack` |
| Route params | Enum associated value (`Hashable`) |
| `modal: 'filter' \| 'add' \| null` | `HomeSheet?` + `.sheet(item:)` |
| Filter draft vs applied | `draftFilter` @Router / `appliedFilter` @Store |
| Repository interface | `protocol TransactionRepository` |
| Mock API / in-memory DB | `InMemoryTransactionRepository` |
| Store actions | `applyFilter()`, `clearFilter()`, `add()`, `delete()` |
| `useMemo` filter in component | `displayedTransactions` computed on Store |
| React Query cache refresh | `load()` after repository write |

## Common Mistakes to Watch For [[CHANGED]]

1. Side effects trong `body` — dung `submit()`, `.onChange`, `.task`
2. Nested `NavigationStack` — chi mot stack o router
3. Pass ca `Transaction` object trong route — chi truyen `id`, lookup tu store
4. Preview thieu `NavigationStack` — `.navigationTitle` / `.toolbar` khong hien
5. DEBUG seed trong `.task` pollute Empty preview — dung `XCODE_RUNNING_FOR_PREVIEWS` guard
6. Filter date: `from` = `>=`, `to` = `<=`
7. Empty state: `store.isEmpty` vs `displayedTransactions.isEmpty` — hai message khac nhau
8. Hardcode repository impl trong Store — inject qua `init`
9. Quen `load()` sau mutation — UI stale
10. Mutate `appliedFilter` truc tiep tu View — dung `applyFilter` / `clearFilter`
11. `@State` owner Store o App — khong tao Store trong View body
