## Key Mental Model Mappings (React Native -> SwiftUI) [[CHANGED]]

| React Native / TypeScript | Swift / SwiftUI |
|---|---|
| `const` | `let` |
| `let` | `var` |
| `interface` / object type | `struct` |
| Service / store class | `final class` |
| In-memory CRUD store | `TransactionManager` |
| `Intl.NumberFormat` | `NumberFormatter` / `Double.asVND()` |
| List item `id` | `Identifiable` |
| `string \| undefined` | `String?` |
| Redux selector | `filter(by:)`, `sortByNewest()` |
| Function component | `struct SomeView: View` |
| Props | stored properties / init (`let title`, ...) |
| `flexDirection: 'row'` + `space-between` | `HStack + Spacer()` |
| Layered badge/watermark | `ZStack(alignment:)` |
| `ScrollView` | `ScrollView` (+ padding tren stack con) |
| `resizeMode="cover"` | `.scaledToFill()` + `.clipped()` |
| `FlatList` / windowing | `LazyVStack` + `ForEach` trong `ScrollView` |
| Sticky section header | `Section` + `pinnedViews: [.sectionHeaders]` |
| `borderBottomRightRadius` | `UnevenRoundedRectangle(bottomTrailingRadius:)` |
| `TouchableOpacity` + `onPress` | `Button(action:)` |
| Callback prop | `@escaping () -> Void` |
| Theme `colors.primary` | `Color("brand.primary")` via `AppColor` |
| SF icon / vector icon | `Image(systemName:)` |

## Common Mistakes to Watch For [[CHANGED]]

1. Dat side effects trong `body` thay vi dung `.task`
2. Co map Flexbox 1-1 sang `VStack/HStack/ZStack`
3. Dung `.frame(width: .infinity)` thay vi `maxWidth`
4. Double padding giua parent/child gay lech spacing
5. Contrast text kem tren gradient/background (dung `.white` / `onBrand`, khong `text.primary`)
6. Anh banner tran ngang: thieu `scaledToFill` + clip
7. `pinnedViews` khong dung `Section` -- khong co tac dung
8. LazyVStack kep nhau khong can thiet -- uu tien VStack co dinh + LazyVStack list
9. Force unwrap `!` trong production code
10. Dead code / duplicate data source (`manyTransactions` vs `sampleTransactions`)
