## Key Mental Model Mappings (React Native -> SwiftUI) [[CHANGED]]

| React Native / TypeScript | Swift / SwiftUI |
|---|---|
| `const` | `let` |
| `let` | `var` |
| `interface` / object type | `struct` |
| Redux/Zustand store | `@Observable` class (Observation) |
| `useState(store)` + mutate | `@State` owner + `@Observable` property tracking |
| Controlled component | `@Binding` — parent `$field`, child `@Binding var field` |
| `React.createContext` + `Context.Provider` | `@State` at App + `.environment(manager)` |
| `useContext(MyContext)` | `@Environment(MyType.self)` |
| `useEffect(() => {}, [])` async | `.task { }` |
| `useEffect(..., [value])` | `.onChange(of:)` |
| Callback prop | Closure `(String) -> Void` |
| Stack Navigator | `NavigationStack` |
| `navigation.navigate("Detail", { id })` | `path.append(.transactionDetail(id: id))` |
| Route params | Enum associated value (`Hashable`) |
| `<Stack.Screen>` registry | `.navigationDestination(for: AppRoute.self)` |
| `navigation.goBack()` | `@Environment(\.dismiss)` / system back button |
| `modal: 'filter' \| 'add' \| null` | `HomeSheet?` + `.sheet(item:)` |
| `presentation: 'fullScreenModal'` | `.fullScreenCover(item: AppCover.self)` |
| Filter draft vs applied state | `draftFilter` / `appliedFilter` on router |
| Storybook + NavigationContainer | `HomePreviewShell` + seeded manager |
| Preview empty state | `TransactionManager()` + skip DEBUG seed in preview |

## Common Mistakes to Watch For [[CHANGED]]

1. Side effects trong `body` — dung `submit()`, `.onChange`, `.task`
2. Nested `NavigationStack` — chi mot stack o router
3. Pass ca `Transaction` object trong route — chi truyen `id`, lookup tu manager
4. Mix `NavigationLink(value:)` va callback cho cung action — chon mot style
5. Preview thieu `NavigationStack` — `.navigationTitle` / `.toolbar` khong hien
6. DEBUG seed trong `.task` pollute Empty preview — dung `XCODE_RUNNING_FOR_PREVIEWS` guard
7. `@State` duplicate o child — child dung `@Binding`
8. Force unwrap `!` trong production
9. Navigate = callback len router; mutate shared state = `@Environment`
10. Filter date: `from` = `>=`, `to` = `<=` — dung operator
11. Empty state: `manager.isEmpty` vs `displayed.isEmpty` — hai message khac nhau
12. Swift struct param order: default param truoc required khi khai bao — goi site phai dung thu tu
