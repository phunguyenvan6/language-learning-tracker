## Key Mental Model Mappings (React Native -> SwiftUI) [[CHANGED]]

| React Native / TypeScript | Swift / SwiftUI |
|---|---|
| `const` | `let` |
| `let` | `var` |
| `interface` / object type | `struct` |
| Service / store class | `final class` |
| In-memory CRUD store | `TransactionManager` |
| `Intl.NumberFormat` | `NumberFormatter` / `Double.asVND()` |
| `moment().fromNow()` | `RelativeDateTimeFormatter` → `displayRelativeDate` |
| List item `id` | `Identifiable` |
| `string \| undefined` | `String?` |
| Redux selector | `filter(by:)`, `sortByNewest()` |
| Function component | `struct SomeView: View` |
| Props / init params | `let` stored property / `init(shouldSeedOnAppear:useNativeList:)` |
| `flexDirection: 'row'` | `HStack + Spacer()` |
| Fragment `<></>` | `Group { }` |
| `ScrollView` + children | `ScrollView` + `VStack` / `LazyVStack` |
| `FlatList` | `List` hoac `ScrollView + LazyVStack + ForEach` |
| `SectionList` header | `Section { } header:` hoac title row khi `embedInList` |
| Sticky section header | `LazyVStack(pinnedViews: [.sectionHeaders])` |
| `TouchableOpacity` | `Button` / `.buttonStyle(.plain)` |
| Callback prop | `@escaping () -> Void` / trailing closure |
| `useState(store)` mutate khong re-render | `@State` + `class` — tam `listRevision`; Phase 3 `@Observable` |
| `setState({})` force update | `listRevision += 1` + `.id(listRevision)` |
| `useEffect` on mount | `.onAppear` |
| Empty list UI | `if transactions.isEmpty { EmptyTransactionsView }` |
| Theme tokens | `AppColor` / `AppSpacing` + Asset catalog semantic colors |
| SF icon | `Image(systemName:)` + `TransactionCategory.systemImage` |
| List row styling | `.listRowInsets` / `.listRowSeparator(.hidden)` / `.listRowBackground` |

## Common Mistakes to Watch For [[CHANGED]]

1. Dat side effects trong `body` — dung `.onAppear` / `.task`
2. Mutate `@State` class manager khong re-render — `@Observable` Phase 3 hoac `listRevision` (tam)
3. Seed trong `body` — duplicate data
4. Map Flexbox 1-1 sang stacks
5. `.frame(width: .infinity)` thay vi `maxWidth: .infinity`
6. Double padding parent/child
7. **List lồng trong ScrollView** — scroll conflict; dung mot root scroll
8. **List Section header** khac custom Text — dung `embedInList` + title row
9. Preview ten sai vs `shouldSeedOnAppear` — "Co giao dich" can `true`
10. `pinnedViews` khong co `Section` — khong sticky
11. Force unwrap `!` trong production
