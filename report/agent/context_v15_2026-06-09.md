## Key Mental Model Mappings (React Native -> SwiftUI) [[CHANGED]]

| React Native / TypeScript | Swift / SwiftUI |
|---|---|
| `const` | `let` |
| `let` | `var` |
| `interface` / object type | `struct` |
| Service / store class | `final class` |
| In-memory CRUD store | `TransactionManager` |
| Redux/Zustand store | `@Observable` class (Observation) |
| `useState(store)` mutate khong re-render | `@State` + `@Observable` class — SwiftUI tracks property reads |
| `setState({})` / bump key | ~~`listRevision`~~ — khong can sau @Observable |
| Controlled component | `@Binding` (Day 2) |
| Callback prop | `(String) -> Void` / trailing closure — `onTransactionTap` |
| `useEffect` on mount | `.onAppear` |
| Redux selector | `sortByNewest()`, `totalAmount()` — doc `transactions` |
| Function component | `struct SomeView: View` |
| Props / init params | `let` stored property / `init(shouldSeedOnAppear:useNativeList:)` |
| `FlatList` | `List` hoac `ScrollView + LazyVStack + ForEach` |
| Theme tokens | `AppColor` / `AppSpacing` + Asset catalog |
| SF icon | `TransactionCategory.systemImage` |

## Common Mistakes to Watch For [[CHANGED]]

1. Dat side effects trong `body` — dung `.onAppear` / `.task`
2. ~~Mutate @State class khong re-render~~ — da hoc @Observable; tranh quay lai listRevision
3. Seed loop cung `Date()` — sortByNewest trông vo nghia; dung createdAt lech hoac tie-breaker
4. Map Flexbox 1-1 sang stacks
5. List long trong ScrollView — mot root scroll
6. Preview `shouldSeedOnAppear` — default `false`; chi preview co data truyen `true`
7. Duplicate model files Week 2/Week 3 lech nhau — sync manager truoc Day 2
8. Force unwrap `!` trong production
